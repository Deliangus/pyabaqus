abaqus.BeamSectionProfile.ArbitraryProfile module

class abaqus.BeamSectionProfile.ArbitraryProfile.ArbitraryProfile(name: str, table: tuple)

Bases: abaqus.BeamSectionProfile.Profile.Profile

The ArbitraryProfile object defines the properties of an arbitrary profile. The ArbitraryProfile object is derived from the Profile object.

  • import section

  • mdb.models[name].profiles[name]

  • import odbSection

  • session.odbs[name].profiles[name]

The first sequence in the table specifies the following: - 1-coordinate of the first point defining the profile. - 2-coordinate of the first point defining the profile. All other sequences in the table specify the following: - 1–coordinate of the next point defining the profile. - 2–coordinate of the next point defining the profile. - The thickness of the segment ending at that point.
