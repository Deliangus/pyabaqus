The Sorption object defines absorption and exsorption behaviors of a partially saturated porous medium in the analysis of coupled wetting liquid flow and porous medium stress.

  • import material

  • mdb.models[name].materials[name].sorption

  • import odbMaterial

  • session.odbs[name].materials[name].sorption

If lawAbsorption*=TABULAR or *lawExsorption*=TABULAR, the *absorptionTable and exsorptionTable data respectively specify the following: - Pore pressure, uw. - Saturation, s. If lawAbsorption*=LOG or *lawExsorption*=LOG, the *absorptionTable and exsorptionTable data respectively specify the following: - A. - B. - s0. - s1.
