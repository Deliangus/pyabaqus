class abaqus.Material.Eos.Eos.Eos(type: abaqusConstants.SymbolicConstantType = 'IDEALGAS', temperatureDependency: abaqusConstants.BooleanType = 0, dependencies: int = 0, detonationEnergy: float = 0, solidTable: tuple = (), gasTable: tuple = (), reactionTable: tuple = (), gasSpecificTable: tuple = (), table: tuple = ())

Bases: object

The Eos object specifies an equation of state model.

  • import material

  • mdb.models[name].materials[name].eos

  • import odbMaterial

  • session.odbs[name].materials[name].eos

If *type*=IDEALGAS, the table data represents the following: - Gas constant, R. - The ambient pressure, pA. If this field is left blank, a default of 0.0 is used. If *type*=JWL, the table data represents the following: - Detonation wave speed, Cd. - A. - B. - ω. (Dimensionless.) - R1. (Dimensionless.) - R2. (Dimensionless.) - Pre-detonation bulk modulus, Kp⁢d. - Detonation energy density, E0. If *type*=USUP, the table data represents the following: - c0. - s. (Dimensionless.) - Γ0. (Dimensionless.) If *type*=TABULAR, the table data represents the following: - F1. - F2. - εcvol. (Dimensionless.)
