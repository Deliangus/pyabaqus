class abaqus.Material.Others.MassDiffusion.Diffusivity.Diffusivity(table: tuple, type: abaqusConstants.SymbolicConstantType = 'ISOTROPIC', law: abaqusConstants.SymbolicConstantType = 'GENERAL', temperatureDependency: abaqusConstants.BooleanType = 0, dependencies: int = 0)

Bases: object

The Diffusivity object specifies mass diffusivity.

  • import material

  • mdb.models[name].materials[name].diffusivity

  • import odbMaterial

  • session.odbs[name].materials[name].diffusivity

If *type*=ISOTROPIC, the table data specify the following: - Diffusivity, D. - Concentration, c. - Temperature, if the data depend on temperature. - Value of the first field variable, if the data depend on field variables. - Value of the second field variable. - Etc. If *type*=ORTHOTROPIC, the table data specify the following: - D11. - D22. - D33. - Concentration, c. - Temperature, if the data depend on temperature. - Value of the first field variable, if the data depend on field variables. - Value of the second field variable. - Etc. If *type*=ANISOTROPIC, the table data specify the following: - D11. - D12. - D22. - D13. - D23. - D33. - Concentration, c. - Temperature, if the data depend on temperature. - Value of the first field variable, if the data depend on field variables. - Value of the second field variable. - Etc.
