## setValues(...)



This method modifies the OdbDataFrame object.



### Required arguments

- *activateFrame*

  A Boolean specifying whether to activate the frame.

### Optional arguments

- *update*

  A Boolean specifying whether to update the model. The default value is ON

### Return value

None.

### Exceptions

None.
