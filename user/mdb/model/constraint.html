Constraint

The Constraint commands define constraints between regions of the model.

Create constraints

class ConstraintModel(name: str, description: str = '', stefanBoltzmann: Optional[float] = None, absoluteZero: Optional[float] = None, waveFormulation: abaqusConstants.SymbolicConstantType = 'NOT_SET', modelType: abaqusConstants.SymbolicConstantType = 'STANDARD_EXPLICIT', universalGas: Optional[float] = None, copyConstraints: abaqusConstants.BooleanType = 1, copyConnectors: abaqusConstants.BooleanType = 1, copyInteractions: abaqusConstants.BooleanType = 1)

Abaqus creates a Model object named Model-1 when a session is started.

Notes

This object can be accessed by:

mdb.models[name]
