Filter

Filter commands are used to create real-time filters of output request data.

Create filters

In Mdb

class FilterModel(name: str, description: str = '', stefanBoltzmann: Optional[float] = None, absoluteZero: Optional[float] = None, waveFormulation: abaqusConstants.SymbolicConstantType = 'NOT_SET', modelType: abaqusConstants.SymbolicConstantType = 'STANDARD_EXPLICIT', universalGas: Optional[float] = None, copyConstraints: abaqusConstants.BooleanType = 1, copyConnectors: abaqusConstants.BooleanType = 1, copyInteractions: abaqusConstants.BooleanType = 1)

Abaqus creates a Model object named Model-1 when a session is started.

Notes

This object can be accessed by:

mdb.models[name]

In Odb

class FilterOdb(name: str, analysisTitle: str = '', description: str = '', path: str = '')

The Odb object is the in-memory representation of an output database (ODB) file.

Notes

This object can be accessed by:

import odbAccess
session.odbs[name]
