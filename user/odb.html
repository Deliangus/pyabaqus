Abaqus Output DatabaseÔÉÅ

class Odb(name: str, analysisTitle: str = '', description: str = '', path: str = '')

The Odb object is the in-memory representation of an output database (ODB) file.

Notes

This object can be accessed by:

import odbAccess
session.odbs[name]
